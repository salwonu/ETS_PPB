**ResepKita**
ResepKita adalah aplikasi Android yang dibuat menggunakan Kotlin dan Jetpack Compose untuk nemenin aktivitas masak-memasak. Saat lagi masak, user butuh aplikasi yang to-the-point dan nggak membingungkan. 
Jadi, aplikasi ini fokus ke pengalaman pengguna yang senatural mungkin, supaya siapa pun yang menggunakan aplikasi ini bisa langsung paham navigasinya tanpa perlu beradaptasi lama.

**Navigasi & Fitur**
Aplikasinya memakai sistem 4 tab supaya pengguna nggak bingung saat menggunakan. Berikut ini beberapa sistemnya:

1. Beranda: Tempatnya ide masakan, kategori, dan resep populer.

2. Cari: Fitur pencarian real-time. Jadi saat pengguna sedang encari resep, hasilnya langsung keluar tanpa perlu tekan tombol enter.

3. Profil: Tempat pengaturan akun dan tampilan (seperti Dark Mode atau Light Mode).

4. Detail Resep: Halaman ini dibuat interaktif. Ada hero image yang besar dan bahan-bahannya juga sudah tertulis sehingga dapat bantu user pas lagi memasak.

**Cara Menjalankan**
Saat menjalankan program ini, saya menggunakan:

- Android Studio: Saya menggunakan Android 14
- JDK: versi 17 ke atas
- Emulator/HP: Bisa pakai emulator bawaan atau HP fisik, namun di sini saya menggunakan laptop

**Langkah-langkahnya**
1. Buka folder proyek di Android Studio.
2. Tunggu proses Gradle Sync sampai selesai. Ini penting supaya library pendukung (kayak Navigation Compose & Coil buat gambar) terpasang semua.
3. Pilih perangkat (emulator atau HP) di device manager.
4. Klik tombol Run di Android Studio (segitiga hijau) atau pencet Shift + F10
